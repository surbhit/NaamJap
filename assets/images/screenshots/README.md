# Screenshots Directory

Place your app screenshots in this directory with the following naming convention:

1. **choose_mantra.png** - Screenshot of the "Choose Your Mantra" screen
2. **my_mantras.png** - Screenshot of the "My Mantras" screen
3. **your_journey.png** - Screenshot of the "Your Journey" analytics screen
4. **mala_completion.png** - Screenshot of the "Mala Completion" screen
5. **spiritual_interface.png** - Screenshot of the spiritual interface/home screen
6. **progress_counter.png** - Screenshot of the progress counter screen

## Image Requirements:
- **Format**: PNG or JPG
- **Recommended size**: 1080x1920 pixels (9:16 aspect ratio for mobile screenshots)
- **File size**: Keep under 2MB per image for optimal web performance

## Notes:
- If an image is not found, a placeholder will be displayed automatically
- Images should be high-quality screenshots from your app
- Make sure images are properly cropped and showcase the app features clearly
